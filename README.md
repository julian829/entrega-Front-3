# entrega-Front-3
La página web del proyecto **entrega-Front-3** consiste en una aplicación desarrollada con **HTML**, **CSS** y **JavaScript** que forma parte de una entrega académica de Front-End. Su objetivo principal es mostrar una interfaz interactiva y dinámica en el navegador, organizada en diferentes secciones visuales y funcionales.

El sitio carga inicialmente un archivo `index.html` que contiene la estructura base del contenido. A través de las hojas de estilo en la carpeta `css/`, se definen los colores, el diseño y la disposición visual de los elementos, utilizando propiedades modernas de CSS como flexbox o grid para lograr un diseño adaptable a distintos dispositivos.

La lógica del sitio está implementada en la carpeta `js/`, donde los scripts gestionan las interacciones del usuario con los elementos de la página. Mediante funciones que utilizan métodos como `getElementById`, `addEventListener`, `.value` y `.textContent`, el JavaScript detecta acciones del usuario, como clics o envíos de formularios, y actualiza el contenido mostrado sin necesidad de recargar la página.

El funcionamiento general consiste en:

1. Cargar la página principal con su diseño e información base.
2. Permitir que el usuario navegue e interactúe con distintos elementos.
3. Registrar los eventos y responder dinámicamente mediante JavaScript, modificando el contenido del DOM según la acción realizada.
4. Mantener una experiencia fluida y visualmente agradable gracias al estilo y la estructura organizados.

La página demuestra los principios fundamentales del desarrollo web front-end: separación de estructura, estilo y comportamiento; uso de código limpio y comentado; y una presentación atractiva y funcional.
